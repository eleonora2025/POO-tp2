Concesionaria de Autos y Motos
Este proyecto simula el funcionamiento de una concesionaria de vehículos (autos y motos) utilizando Java, Spring Boot, Maven y Lombok.
El programa está diseñado siguiendo principios de Programación Orientada a Objetos y utiliza interfaces, clases abstractas, herencia, Comparable, Collections y Streams.

Estructura del proyecto

entidades: Clase padre: Vehiculo; Clases hijas: Auto, Moto (ambas heredan de Vehículo)

interfaces: interfaz: IVehiculo class: Implementación

tests: Test
